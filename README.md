# MeetSync 📅

MeetSync is a collaborative calendar application that helps groups find common available days for meetings, trips, workouts, or any shared activity.

Instead of exchanging countless messages, users can simply mark their availability with colors on a shared calendar.

> 🚧 This project is currently under development.

---

## ✨ Features

- 📅 Interactive monthly calendar
- 🎨 Multiple color selection for different users
- 👥 Up to 4 participants per day
- 💾 Local storage support (availability is saved)
- 📆 Dynamic calendar with month navigation
- ⚡ Built with React + TypeScript + Vite

---

## 🚀 Tech Stack

- React
- TypeScript
- Vite

- LocalStorage
